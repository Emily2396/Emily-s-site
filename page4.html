After some more time, it became clear that they needed to include more literary works as computer programs has started to came about, so in 1992 It was extended.[1]. The copyright law now protects:

1.6 Describing access issues that need taking into account

When designing a website that is being used by a whole county, it must be accessible for all. A good example of a website that needs to be accessible to all is the UK government digital service. As this website could be used by anyone in the UK, it has multiple changes that you can implement onto the site, that includes:

  • Being able to zoom in on the text on the webpage so anyone is able to read it
  • Screen reader. There is a large button on a page that when clicked it will read out the whole page to the person
  • Speech recogniton. Being able to use you voice to navigate the site
  • Change the colour/font of the page to suit the persons eyes better
  • and more
It is important to remember of every person who might be visiting your webpage. If your page isn't user-friendly it can discourage people from visiting the site if they feel they can not understand and/or use the site.

1.7 Describe which are the appropriate filetypes for websites

While creating a website you will come across many different filetypes that you could use. Chosing which ones you want to use and why can sometimes make no difference or all the difference, and knowing which ones do/dont is important. I have listed some one the filetypes I have used and others I could have.

  • PNG. Portable Network Graphics. This is used for high-quality images. This file type helps keep the quality of a photo so when you copy and paste it will retain its quality.PNG includes images without a background too. The only downside of PNG is that the images can take up a lot of storage and so increase the webpages loadtime.
  • GIF. Graphics Interchange Format, is a small looping animation. Basic animations can make a website a little different, like simple designs for businesses can be a nice addition but a GIF that isn't simple can display poorly and so making the website look like professional.
  • PFD. Portable Document Format. This file is used for online documents and printing. A PDF can contain different elements such as photos, tect, audio and more. This filetype is good as it always keeps its formatting no matter what device it is on but you must have a PDF reader to be able to view it.
  • MP4. Moving Picture Experts Group. This is a video. This kind of file can contain, video, audio, and subtiles. The compression format works so well that it can create a lightweight file that is easily loaded on websites.
  • MP3. Moving Picture Experts Group(MPEG) Audio Layer-3. this is an audio file saved in a compressed audio format. A mp3 would need to have a Media player to be able to add on to a webpage.
